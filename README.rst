===============================
rio - minimalist http server 
===============================

rio is a minimalist http server

this project is a work in progress

Building
--------

Open terminal::
    
    make

Tesing
--------

Install CUnit (http://cunit.sourceforge.net)
Open terminal::
    
    make test

Running
-------

Open terminal::
    
    sudo ./rio
